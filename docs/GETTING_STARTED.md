# Getting Started with STX Tools

## Overview
STX Tools provides utility functions for working with the Stacks blockchain.

## Installation

```bash
npm install
clarinet integrate
```

## Quick Start

1. Clone the repository
2. Install dependencies with `npm install`
3. Run tests with `clarinet test`

## Usage Examples

See the `tests/` directory for comprehensive usage examples.

## API Reference

Refer to the contract source files in `contracts/` for the complete API.
